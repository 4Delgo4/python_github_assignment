# python_github_assignment

Title: 2-d shape area calculator

Short Description: The program calcualtes the area of 4 shapes, Square, Rectangle, Circle, and a Triangle. It asks for the shape they want to callculate with using . upper() to capitalize their input to reduce the chance of the code not running. Then it is followed by if, elif, and else statements to calculate the area of their shape based on the name of the shape. Also if they enter an invalid input it returns a messge to reduce the chance of an error. 

How to run: enter one of the shapes listed: Square, Rectangle, Triangle, Circle. Then enter the unit of measurement: Centimeters(CM), Inches(IN), Feet(ft). etc. And lastly enter the size of the variable it prompts you in based on your measurement
